# MovieHub — Frontend Angular

Frontend de la plataforma MovieHub, desarrollado con Angular 22 standalone.

## Estado actual

Proyecto scaffolded con Angular CLI. Pendiente de desarrollar:
- Componentes de listado, detalle, formularios
- Servicios de consumo de API
- Angular Material
- Autenticación JWT
- Tests con Vitest

## Comandos

```bash
ng serve          # Servidor de desarrollo → http://localhost:4200
ng build          # Build de producción
ng test           # Tests unitarios (Vitest)
```
